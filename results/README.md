# Result CSVs

These CSV files mirror the manuscript tables and are loaded by `notebooks/04_generate_paper_tables_figures.ipynb`.

The current manuscript uses these table numbers:

- Table 1: class distribution
- Table 5: classification performance
- Table 6: RQ1 perturbation stability
- Table 7: LIME stochasticity
- Table 8: RQ2 model-variation consistency
- Table 9: RQ3 F1-XAI correlation
- Tables 10-11: RQ4 fidelity, split by dataset in the manuscript and combined here
- Table 12: statistical tests
- Table 13: XAI-Score
- Table 14: adversarial robustness

Compatibility alias files are included for older task notes:

- `table_11_statistical_tests.csv` duplicates `table_12_statistical_tests.csv`
- `table_12_xai_score.csv` duplicates `table_13_xai_score.csv`
- `table_13_adversarial.csv` duplicates `table_14_adversarial.csv`
